# Cidadão Ativo

## English

### About the Project

**Cidadão Ativo** is a civic participation application designed to strengthen the relationship between citizens and local governments.  
The platform allows residents to register, track, and follow up on reports related to **urban infrastructure** and **public safety** issues in their city.

The project aims to improve transparency, efficiency, and responsiveness in municipal public services through digital participation.

### Project Status

This project is **under development** and is being carried out **individually and privately**.

- The source code is not publicly available.
- This repository is intended to present the **project overview and documentation**.

### Technology Stack

- **Frontend:** React Native with TypeScript  
- **Backend:** Currently under discussion and definition  

### Target Audience

The application is intended for **municipal governments in the state of Pernambuco, Brazil**, focusing on:
- City halls and public administrations  
- Municipal infrastructure and public safety departments  

### Availability

The project has not been publicly released yet.  
Access, pilot programs, and partnerships are evaluated on a case-by-case basis with interested municipalities.

## Português

### Sobre o Projeto

**Cidadão Ativo** é um aplicativo voltado à participação cidadã, com o objetivo de fortalecer a relação entre a população e o poder público municipal.  
A plataforma permite que os cidadãos registrem, acompanhem e monitorem ocorrências relacionadas à **infraestrutura urbana** e à **segurança pública** em sua cidade.

O projeto busca promover mais transparência, eficiência e agilidade na prestação dos serviços públicos por meio da participação digital.

### Status do Projeto

Este projeto está **em desenvolvimento** e é conduzido de forma **individual e privada**.

- O código-fonte não está disponível publicamente.
- Este repositório tem a finalidade de apresentar a **visão geral e a documentação do projeto**.

### Tecnologias Utilizadas

- **Frontend:** React Native com TypeScript  
- **Backend:** Em fase de discussão e definição  

### Público-Alvo

O aplicativo é destinado a **prefeituras e administrações municipais do estado de Pernambuco**, com foco em:
- Gestões municipais  
- Secretarias e setores de infraestrutura e segurança pública  

### Disponibilidade

O projeto ainda não possui lançamento público.  
Parcerias, testes piloto e disponibilização para uso institucional são avaliados de forma individual junto às prefeituras interessadas.
